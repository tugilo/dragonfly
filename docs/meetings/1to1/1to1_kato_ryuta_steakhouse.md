## ■ 事業理解（第1回121で確定）

### 1. 本業「事務屋」（コア）

| 項目 | 内容 |
|------|------|
| **定義** | クライアント企業の **スタッフとして内部に入り込み**、総務・人事・経理・営業・広報・**補助金・助成金**・資金調達・**クレーム対応** など **ありとあらゆる事務** |
| **ターゲット像** | **フライパンを握り続ける** 飲食オーナー、**現場職人** など — 事務ができない・やれない経営者 |
| **運営モデル** | 最初に **「散らかった部屋を片付ける」** ように整頓 → その後 **事務代行スタッフ** や **障がいのある方**（うつ・精神障害等）へ **引き継ぎ** |
| **ポジション** | **「無色透明の事務員」** — 全方向に敵を作らない。既存経理がいても **補助金・広報** 等で入れる |
| **税理士事務所** | **阿久津公一税理士事務所** スタッフとしてBNI登録。**建設業専門** 支援は継続 |

### 2. 飲食・食材ビジネス

#### ステーキハウス（BNIカテゴリー）

- **参加理由:** 税理士事務所 **顧問先オーナー** が趣味でステーキハウス運営 → **事務所スタッフ** としてBNI参加
- **加藤の関与:** **店舗に常駐しない**
- **主目的:** **FC展開** — 飲食店を出したい人・FC希望者を探し、**自社仕入れ先（食材サプライヤー）の販路拡大**

#### 仕入れ構造・利益率（具体例）

| 商品 | 仕入れ | 販売・モデル |
|------|--------|--------------|
| **タミー食品** カレー（モスバーガー・ゴーゴーカレー向け製造） | **250円** | 飲食店 **1,000円** |
| **元気豚**（肉屋） | **130円**（焼くだけ） | 一般市場500円の肉を店舗1,000円／祭り等500円 |
| **魔王（焼酎）** | **5,000円**（市場1.3〜1.5万円） | **1,000円×36杯＝36,000円**、売れた後 **+6,000円** |

#### 子ども食堂

- **250円カレー** を先に **無償提供** → 大人向け **1,000円** 販売で **600円以上** を子ども支援に
- **全国50社以上** 提携

#### 補助金・助成金

- 新規飲食店立ち上げの **補助金・助成金申請** も対応（**インシャルステート** 等）
- **一人飲食店経営は避け、チームを作る** ことを推奨
