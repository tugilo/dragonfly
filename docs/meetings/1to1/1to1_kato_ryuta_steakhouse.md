### 1. 事務屋（コア）

クライアント企業に **スタッフとして入り込み**、バックオフィス全般を担当。

| 領域 | 内容 |
|------|------|
| **担当範囲** | 総務・人事・経理・営業・広報・補助金・資金調達など **全般** |
| **スタンス** | 税理士を変えなくても **「会社の一員のかとちゃん」** として動く（Slides・121一致） |
| **税理士事務所** | **阿久津公一税理士事務所** 所属スタッフとして継続参加 |
| **顧問先支援** | 肉屋・カレー会社など **販路支援** も兼務 |

| 項目 | 内容 |
|------|------|

| **食材調達** | **魔王**（仕入れ5,000円 → 1杯1,000円×36杯）など **調達・原価設計** も担当 |
| **BNIカテゴリー** | 名簿 **ステーキハウス** はこの飲食・FC・食材軸に対応 |
