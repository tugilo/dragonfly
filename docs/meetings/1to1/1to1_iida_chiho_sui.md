## ■ 事業理解（彗 sui）

### 何を提供しているか

- **視える・聴こえる** リーディングに、命術・手相を掛け合わせた鑑定。
- 一時的アドバイスではなく **「本質的な流れ」** と **「現実的な行動」** の両方を伝える。
- 経営者顧問実績あり。スピリチュアルに偏らず **売上・事業・人間関係** に直結する判断サポート。
- 鑑定後は「**決断スピードが上がる**」「**迷いが消える**」と言われることが多い。

### 定量・実績（G.A.I.N.S.・シートより）

| 区分 | 内容 |
|------|------|
| **鑑定実績** | **1000人以上** |
| **顧問** | 法人顧問鑑定 **5社** |
| **その他** | プロスポーツ選手鑑定 |
| **SNS** | Instagram 占いライブ **毎朝 8:30**、**丸3年** 継続 |
| **顧客獲得** | 紹介・SNS（Instagram/Threads）・イベント・LINE 登録後案内が多い |
| **事例** | 鑑定後2ヶ月で **売上3,800万** 達成した事例あり（シート記載） |
| **継続** | 単発だけでなく **継続契約**（年間伴走型顧問）につながるケースが多い |

### 理想顧客・紹介条件（ONE to ONE シート）

**こんな人がお客様**

- 経営・仕事で決断する立場の方
- 考えすぎて動けなくなる方、選択に確信を持ちたい方
- 占いに興味はあるが活用法がわからない方
- 売上・事業の流れを加速させたい方、人生の転機（転職・独立・結婚等）
- 特に「**なんとなくではなく、確信を持って進みたい**」方

**話の切り出し（第三者向け）**

- 「経営や仕事の判断に占いを使うってどう思いますか？」
- 「決断に迷うことってありますか？」
- 「実は、経営者向けに判断が速くなる占いをやっているんです」
- 「占いって当てるだけじゃなくて、意思決定に使えるって知ってましたか？」

**質の高いリファーラル**

- 決断や人生に本気で向き合っている方
- 価格ではなく **価値** で判断できる方、行動する意志がある方
- 占いをエンタメではなく **ツール** として使いたい方

**不適切なリファーラル**

- 答えだけ求めて行動しない方、依存的になりやすい方
- 価格重視で比較する方、冷やかし・興味本位だけの方
