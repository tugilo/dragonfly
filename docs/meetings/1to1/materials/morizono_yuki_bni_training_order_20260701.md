## トレーニング修了と121の質（次廣の実感）

BNIメンバーの中には、**相手がトレーニングをどこまで終えているか** を（意識的・無意識的に）見ている方も多い。これは「上から目線」というより、**121の時間を有効に使いたい** という文化に近い。

| 観点 | トレーニング前 | トレーニング後 |
|------|----------------|----------------|
| **自己紹介** | 長い・散漫になりがち | 30秒/60秒の **型** で要点が伝わる |
| **GAINS** | 聞いたが整理できない | **ワークシート** の意図が分かり、深掘りできる |
| **紹介の依頼** | 「紹介してください」で終わり | **誰を・どんなサインで・どう** 伝えてほしいか具体化 |
| **相手の時間** | 説明に時間を取られがち | **聞く・返す・次アクション** のバランスが取れる |
| **相手から見た印象** | 「まだBNI慣れてないな」 | 「この人、121の型を知ってる」→ **信頼・再121** につながる |

**実際にスキルは上がる。** MSPでVCP・GAINS、キースキルズ1to1で紹介の一言、ウィークリーで25秒4要素—— **受けるたびに「話す型」が体に入る**。121本数を増やす前に土台を作ると、**1回1回の密度** が変わる。

**森園さん向け:**

- いま **TLT受講中** は正しい順番。Step 1（MSP）・Step 6（KS・1to1）を早めに終えると、7月後半からの **121本格化** の質が一気に上がる
- 相手が「MSP終わった？」「キースキルズ受けた？」と聞くこともある → **恥ずかしいことではなく、共通言語がある証拠**
- 121を **週1** にしながら **Step 4以降のNCAS WS** も並行すると、BOR＋FB＋121で **他章メンバー** との接点も増える

**次廣の体感:** 3月入会直後の121と、4月にトレーニングを一通り終えた後の121では、**相手の反応・紹介の具体性・再アポの取りやすさ** が明らかに違った。

- **121の種** が一気に増える — 受講＝ネットワーク拡大

## 次廣の実践フロー（WS → 121）

NCAS の WS（特に Step 4以降）では、次廣は **必ず** 次の流れでつなげている。

```
① WS中 … BOR で話した相手の名前・チャプター・専門分野が分かる画面を **スクショ**
② 当日〜数日以内 … Facebook で **フレンド申請**（「今日の○○WSで一緒でした、次廣です」）
③ 承認後 … **121** を打診（Zoom・対面どちらでも）
```

| ポイント | 内容 |
|----------|------|
| **スクショ** | BOR中の **Zoom参加者表示** や **自己紹介スライド** など。後から名前を思い出す保険 |
| **FB** | BNIメンバーはFB連絡が多い。申請メッセージに **WS名・チャプター名** を1行入れる |
| **121** | WSで話した続き・GAINS深掘り・相互紹介。 **売り込みではなく信頼構築** |
| **タイミング** | 熱が冷めないうちに。遅くとも **1週間以内** が理想 |

**森園さん向け:** Step 4以降を受け始めたら、この流れを **最初から習慣化** すると、4Stellaの紹介網が DragonFly 外にも広がる。増本さん・平岡さん級の接点も、WS経由で増やしやすい。
