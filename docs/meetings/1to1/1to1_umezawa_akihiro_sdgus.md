## ■ 事業理解（SDGusサポーターズ）

### 何を提供しているか

- **日本JC公認SDGsアンバサダー**（全国48名）として、企業・地域諸団体向けに **SDGs導入コンサル**・**セミナー** を開催。
- 核心商品は **TGX（脱炭素経営シミュレーションゲーム）** を使った **体験型研修**。脱炭素の知識研修ではなく、**企業の経営判断を体験する場** として設計。
- 参加者が自社の事業環境を踏まえて意思決定することで、環境への取り組みを **「理念」ではなく「経営課題」** として考えられるようになることを重視。
- **GX（グリーントランスフォーメーション）** — 会社のしくみや働き方を変え、地球と経済の両方によい影響を与える取り組みも推進。

### TGX（第1回121で確認）

| 項目 | 内容 |
|------|------|
| **正式名称** | 脱炭素推進カードゲーム **TGX** |
| **形式** | **10時間分の座学** を **約3時間** で体感できる **ビジネスカードゲーム** |
| **独自性** | **日本唯一** の企業研修特化型カードゲーム |
| **進行** | 4〜5人で1社を作り、サイコロでコマを進め **脱炭素プロジェクト** を実施。売上・コスト・CO₂排出量が変動 |
| **人数** | 最低 **4人**〜最大 **20人** 程度 |
| **所要** | 体感 **2〜3時間**（「あっという間」の感想多数） |

### 脱炭素経営の背景（121）

- 環境省「未来の天気予報」— 温暖化進行で2100年8月に日本各地 **40℃超** の可能性
- 高校野球の **昼試合廃止** 等、スポーツ分野にも影響拡大
- **改正省エネ推進法** — 大企業が中小製造業に **CO₂見える化・削減** を取引条件として要請開始
- 現場課題：カタカナ・英語が多く現場に伝わりにくい。商工会議所アンケートで **約半数以上** が「マンパワー・ノウハウ不足」

### ビジネスチーム（「知る・測る・減らす」）
