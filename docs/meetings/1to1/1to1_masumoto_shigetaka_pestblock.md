## ■ 実施後サマリー（第1回 2026-06-12・履歴）

### 主な成果（ピタカット × P-Link）

- **プロテクトラボ** の新事業 **ピタカット** を、既存 **P-Link** に組み込む方針で協議。**別システムは不要** で、現行の商品登録・管理・注文の仕組みを拡張すれば足りる、という理解で一致した。
- 害虫ブロック（代表特約店→取扱店→施工）とは別に、ピタカット専用の **「販売代理店」** 役割を P-Link 上に新設する方向。専用ログイン・注文履歴・統計・お知らせの送信先分岐を想定。
- 既存基盤の延長として **20万〜30万円程度** で実装可能との見通しを次廣が共有。増本側も現実的なレンジとして受け止めた。
- 併せて P-Link 本体の改善（変更履歴・Excel 廃止・二重登録防止）も合意。こちらは害虫ブロック運用の信頼性向上にも直結。

### 補足（同席で触れた話題）

Zoom 要約には DragonFly チャプター運営（WEBマスター候補・来期三役・**法被** 等）の話題も含まれていたが、**本 1to1 の主目的はプロテクトラボ案件** である。BNI 関連は [§ DragonFly（補足）](#-dragonfly補足同席で触れた話題) を参照。

### 決定事項

#### ピタカット × P-Link 組み込み（本件）

| 項目 | 内容 |
|------|------|
| **販売代理店制度** | 代表特約店・取扱店とは別の **「販売代理店」** 役割を新設。ピタカット専用の管理体制とする |
| **価格体系** | **スタンダード**（1ケース6本）: 仕入 3,500円・販売 5,500円／**チャレンジ**（16ケース96本）: 仕入 3,000円 |
| **ネット販売** | 値崩れ防止のため、販売代理店のネット販売は **原則禁止** の方向で検討 |
| **システム利用料** | 新規登録費 **20,000円**、年間更新料 **20,000円**（または **30,000円**）を販売代理店から徴収 |
| **インボイス** | 販売代理店は **インボイス登録事業者** であることを契約条件とする |
| **P-Link 上の実装方針** | 既存の商品登録・管理を拡張。**販売代理店用の別ログイン**、ピタカット専用の注文履歴・統計。害虫ブロック注文と **同一管理画面＋フィルタ切替** の案を検討 |
| **組み込みの前提** | 新規システムは作らず、P-Link を **プロテクトラボの販売基盤として一本化** する |

|------|------|
| **変更履歴** | 契約点管理画面の下部に変更履歴セクションを追加し、誰がいつ何を変更したかを記録・表示 |

#### ピタカット × P-Link — 組み込みの検討（本題）

**なぜ P-Link 拡張か**

- プロテクトラボは害虫ブロックFC向けに **P-Link** を既に運用中（代表特約店・取扱店・注文・統計）。
- ピタカットは **別商品・別販売チャネル**（飲食店向け液剤、販売代理店制度）だが、本部が一元管理する構図は同じ。
- 増本の意向は **Excel・別ツールを増やさず、P-Link に載せたい** こと。次廣も既存基盤の拡張で足りると判断。

**組み込み方針（合意の方向）**

| レイヤ | 害虫ブロック（既存） | ピタカット（新規・P-Link 上に追加） |
|--------|----------------------|--------------------------------------|
| 役割 | 代表特約店 → 取扱店 → 施工 | **販売代理店**（第3の役割として新設） |
| ログイン | 既存の階層別 URL | **販売代理店専用ログイン**（代表・取扱店とは別） |
| 商品 | 施工ブロック等 | ピタカット（ケース単位の液剤） |
| 注文管理 | 既存フロー | 同一画面で管理し **フィルタで切替** する案 |
| 本部機能 | 統計・契約点管理 | ピタカット専用の注文履歴・統計・お知らせ（送信先: 代理店のみ／全体 等） |
| 技術的参照 | — | 既存の **本部直注文** の概念を応用可能 |

**組み込み可能と判断した根拠**

- 商品マスタ・注文・権限の枠組みが P-Link に既にある。
- 新規は「販売代理店」ロールとピタカット商品の追加、画面・統計の分岐が中心で、**20万〜30万円** レンジの拡張開発に収まる見込み。
- 二重登録防止は実装済み。変更履歴・Excel 廃止を足せば、ピタカット追加前に本体の運用品質も上がる。

**組み込まない／別途検討としたもの**

- 飲食店向け **決済代行つきの直接オンライン販売** — 値崩れリスクが高く、第1フェーズでは見送り方向。
- **LP から P-Link 直結で決済完結** — 構想はあるが、まずは **LP は宣伝のみ → 購入は販売代理店へ** が最もシンプル。

|------|----------------------|------------------------|

#### リスク（ピタカット × P-Link）

- **成否:** 販売代理店をどれだけ増やせるか。システムはその土台

- **MTLR の矛盾感:** 倉持 賢一（スコア **63**）、越賀 淑恵（**92**）、平岡 国彦（**81**）など、MTLR 上で高得点帯に達していない（または **スコア100・ゴールドクラブメンバー** 実績がない）メンバーが、**アンバサダーサポート** として他チャプターを指導している状況への違和感
- **用語の区別:** MTLR の **総合スコア**（70+ が目安、増本は **100** 等）と、BNI の **100点メンバー**／**ゴールドクラブメンバー**（GCM）達成は別指標。会話では PDF を見ながら両方の文脈が混ざっていた可能性あり
- **100点メンバー減少:** NE リージョンで以前100人以上いた100点メンバーが現在34人に（**100点までの道のり** 系の話題）
- **LDST:** **太田 一誠** は「長くて意味があるのか」と疑問視。次廣は実際に参加し建設的な議論があると確認
- **WEBマスター属人化:** 倉持が自分にしかできない運用にしてしまい、誰も引き継げない状態
