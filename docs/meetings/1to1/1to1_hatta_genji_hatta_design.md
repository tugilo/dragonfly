## ■ AI活用に関する議論（第1回）

### 双方の共通認識

| 観点 | 内容 |
|------|------|
| **品質の芯** | AIは **60〜80点** を出せるが、**100点** に仕上げるには **プロの手** が必要 — **双方一致** |

| **工数** | AI活用で工数短縮 → **単価調整** や **次案件への迅速移行** が可能（八田指摘） |

### デザイン分野でのAIの現実（八田）

| 観点 | 内容 |
|------|------|
| **修正の不安定さ** | 一発目の生成は得意だが、**細かい修正指示** への対応が不安定。プロンプトを少し変えるだけでデザインが大きく変わる |
| **反復** | **10回程度** のやり取りでようやく1つ完成するケースも |
| **使える領域** | **人物が映らないイラスト** 等はAI生成活用可。**著作権** には注意 |
| **Adobe Firefly** | 「**クオリティが低い割に高額**」。バージョンアップのたびに **バグ増加** と否定的評価 |
| **参考** | **Pinterest** 等のデザイン参考サイトの方が **トレンドキャッチ** に有効 |

### AIが業界に与える影響

| 話者 | 内容 |
|------|------|
| **八田** | 「AIに仕事を取られる」懸念から **積極的活用** へ意識転換。淘汰されても仕方ない部分はあるが、**経験に基づく判断力** が最終的な差別化要因 |
