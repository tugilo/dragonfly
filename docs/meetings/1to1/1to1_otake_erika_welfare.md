## ■ 事業理解（大竹絵理香）

### 二本柱の事業

絵理香さんは **労働収入（ネイル）** と **権利収入（福利厚生事業）** の二本立てで活動し、**時間的・経済的な自立** を最終目標としている。

#### ① ネイル事業（労働収入）

- 一軒家の一室で自宅ネイルサロンを運営。
- ネイル歴6〜7年目。独学・最低限の費用でネイリスト検定（1級）・ジェルネイル検定（上級）を全取得。
- サロン勤務 → 店長 → 昨年（独立年）に独立。
- 顧客情報はすべて **記憶で管理**（カルテなし）。お客様に「ふわっと匂わせて」前回の情報を引き出す運用。

#### ② 家族も使える福利厚生事業（権利収入）

| 要素 | 内容 |
|------|------|
| **仕組み** | 個人・中小企業向けの福利厚生サービスを広めるネットワーク型事業。ガソリン割引など日常生活に直結するサービスを含む |
| **収入モデル** | 権利収入（手数料収入）。直接声をかけた人数が4〜6人程度でも、紹介が倍々に広がることで **月最大500万円** の収入上限に達した事例がある |
| **上限設計** | 1人あたり月500万円で打ち止め。「とんがり山」ではなく「台形」型の収益分布を目指す設計で、一人勝ちを防ぐ思想が根底にある |
| **理事長の理念** | 社会貢献・雇用創出を重視。ネットワークビジネスの悪いイメージを変えたい意図 |
| **現在の課題** | サービス内容が充実段階。人が増えるほどサービスが拡充される構造 |
| **パートナー募集** | 同じ収入を目指して一緒に動くパートナーを積極的に探している |

### 独立・事業の動機

- 経済的自立と、子どもへの余裕ある支援。
- 夫の収入に依存せず、自分の判断で動ける環境を作りたい。
- 旦那の扶養から外れた際に **福利厚生の対象外** となり、家族なのにサービスを受けられなくなった経験から、「当たり前ではない」ことを知ったのが福利厚生事業の原点。
