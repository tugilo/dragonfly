## パティシエを目指したきっかけ

| 項目 | 内容 |
|------|------|
| **転機** | 父に塾通いを頼んだが断られた瞬間、進学の夢を諦め「将来の夢は何にしようか」と考え始めた |
| **きっかけ** | 当時流行のアニメ『**夢色パティシエール**』の影響。飛行機欠航で東京滞在中に三男から職業を聞かれたことも契機 |
| **進路情報** | 「鹿児島情報高校にパティシエコースがある」と教えてもらい、パティシエを目指すことを決意 |
| **本質的な理由** | お菓子が好きだからではなく、「**葬式でも結婚式でも、最後は必ず笑顔になる場所に寄り添える仕事**」という合理的な思考 |
| **母親との接続** | 母を小学3年生で亡くしたことが「笑顔に立ち会える仕事」への志向に繋がっている |
| **当初の想定** | 身長が高くないため「10年続ける職業ではない、3年で辞めるだろう」と入学時点で想定 |

## 「反応率にこだわるウェブデザイン」の背景

| 項目 | 内容 |
|------|------|
| **命名の理由** | マーケティング系スクール出身のため、ヒートマップツールなど反応率を数値で把握する手法を学んでいた。「ただ作るだけの人間」にならないよう、マーケティング視点を取り入れたデザインを志向 |
| **横山さんの洞察** | 「お菓子作りも含め、**人の反応を大切にする** というルーツがウェブデザインの理念に繋がっている」と指摘。他のデザイナーにはない強みとして評価。「**案件があればぜひ紹介したい**」と発言 |

## AI時代における今後のビジョン

| 項目 | 内容 |
|------|------|
| **意志** | 5年・10年後もウェブデザインの領域に関わり続けたい |
| **方針** | AIがデザイン制作を担う部分が増える中で、人間にしかできない領域（**サイト設計・仕様策定・数値的な詰め**）に特化 |
| **目指す姿** | ワイヤーフレームからレター（ライティング）まで一貫して担当し、「自分が作ったサイトでどれだけ結果・反応が取れるか」を体現できる人材 |
| **戦略** | 「**AIに食われないよう、AIが不得意な部分を突出させる**」 |
| **小中さんのコメント** | 「今はAIがコードを書いてくれるので英語ができなくてもエンジニアになれる」 |
