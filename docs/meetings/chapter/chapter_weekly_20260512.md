## 教育コーナー：エレベーターピッチ

**担当:** 竹内駿太（エデュケーションコーディネーター）  
**テーマ:** エレベーターピッチ

商品スペックではなく、**顧客の問題と解決後の未来**を短く伝えることが重要だと共有された。

### 要点

| 観点 | 内容 |
|------|------|
| 主役 | 自社商品ではなく、顧客が抱えている問題 |
| 悪い例 | 「私は最新のAIツールを販売しています」では、聞き手は紹介先を思い浮かべにくい |
| 良い例 | 「残業に悩む経営者の時間を1日2時間作り出す仕事をしています」なら、相手が前のめりになりやすい |
| 信頼の裏付け | 「成約率30%改善」「500世帯の家計改善」など、具体的な数字で実績を伝える |
| 行動依頼 | 「誰かいい人」ではなく、地域・規模・立場・悩みまで絞って依頼する |

### 今週のワーク

ワントゥーワンや名刺交換の場で、自分のビジネスを **「誰の悩みを解決し、どんな実績があるのか」** という形で30秒以内に話す練習をする。

### 次廣への落とし込み

「AI業務改善システム構築」ではなく、以下のように話す方が紹介されやすい。

> 人を増やしたくないけれど、日報・見積・問い合わせ対応・毎月の集計に時間を取られている中小企業の社長に、LINE・Web・AIを使って、確認だけで回る仕組みを作っています。

## メインプレゼンテーション

### 大竹絵理香 — 家族も使える福利厚生

#### 背景

日本企業の99.7%を占める中小企業では、大企業のような福利厚生を提供しづらい。大竹さん自身も、以前勤務していたネイルサロンで福利厚生が整っておらず、それを理由にスタッフが辞めていく場面を見てきた。

#### サービス内容

| 項目 | 内容 |
|------|------|
| 提携先 | 全国600社、10,000種類以上のサービス |
| お祝い金 | 出産・結婚・入学など人生の節目に最大30,000円 |
| 保険 | 個人賠償責任保険 500,000,000円まで付帯、示談交渉サービス付き |
| 対象領域 | 電気・ガス、日用品、レジャー、慶弔など |

#### ビジネスモデル

- 会員数が増えるほどサービス内容が良くなる仕組み。
- 利用するだけでなく、会社の事業として収益化・雇用創出につなげられる。

#### 紹介希望先

- 保険代理店
- 財務コンサル
- コミュニティ運営者
- 新しい収入の柱を探している方

### 清原佳彩美 — サロン向け育毛機器・商材販売

#### プロフィール

仙台で美容サロンを経営しながら、シングルマザーとして2人の子どもと2匹のダックスを育てている。20代でまつげパーマに出会い、美容が人の人生を変える力を持つと実感。副作用や肌トラブルが少なく結果の出る商材「ノベルラッシュ」を開発し、1年半で全国600店舗以上へ導入した。

#### N2プラスの特徴

| 観点 | 内容 |
|------|------|
| 施術 | 針を刺さない、触れない、ダウンタイムなし |
| 技術 | 高圧高速で美容成分を頭皮の奥へ届ける |
| 時間 | 月1回、10分程度の施術 |
| 用途 | 育毛、フェイシャルリフトアップ、スカルプ、ボディまで対応 |

#### 実績・症例

- 1回の施術で変化が見られた症例。
- 4回・4ヶ月後に大幅な改善が見られた症例。
- 若い女性の円形脱毛症が3回の施術後、半年で回復した症例。
- 病院で改善困難だった抜け毛が、1回・1ヶ月で産毛が生えた症例。

#### ビジネスメリット

- 技術不要で導入しやすい。
- 10分以内の施術で高単価メニュー化しやすい。
- 美容業界の長時間労働・低単価問題を解決し、利益が残る美容を目指せる。

#### 今後の予定・紹介希望先

2026-05-18 からクラウドファンディング開始予定。

- 美容室オーナー
- 美容系サロンコンサル
- 新しい高単価メニューを探しているサロン経営者
