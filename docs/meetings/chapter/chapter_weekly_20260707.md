## 4. ネットワーキング学習コーナー

**担当:** 竹内駿太（エデュケーションコーディネーター）  
**テーマ:** バスケットボールのショットクロック理論と LCD メソッド

竹内駿太さんより、バスケットボールの「ショットクロック」を比喩に、ウィークリープレゼンで具体的なリファーラル依頼を出す重要性が共有された。

| 観点 | 内容 |
|------|------|
| **ショットクロックの比喩** | バスケットボールのオフェンスが24秒以内にシュートを打たなければターンオーバーになるように、ウィークリープレゼンでも時間内に「具体的なリファーラル要求」というシュートを打つ必要がある |
| **LCDの意味** | Least Common Denominator（最小共通分母）。自分のビジネスを、これ以上分解できない一つの具体的な要素まで切り取ること |
| **悪い例** | 「資産形成のプロです。物件も投資も事業承継もすべてやります」 |
| **良い例** | 「今日は、法人税の支払いに頭を悩ませている設立3年目の社長の課題解決にだけフォーカスします」 |
| **拡大目標との連動** | 現在51名から75名体制を目指しており、75名になると紹介パスコースは2,775通り、現在の約2.2倍に拡大する |

75名全員が LCD に基づいて毎週明確な「シュート」を打てば、チャプター全体の売上スコアボードを大きく伸ばせるというメッセージで締められた。
