## 5. ネットワーキング学習コーナー（エデュケーション）

**担当:** 竹内駿太（エデュケーションコーディネーター）  
**テーマ:** 「ピックアンドロール」戦術からパワーチームの極意を学ぶ

バスケットボールのピックアンドロール（スクリーンプレイ）を BNI 活動に置き換えた解説が行われた。

| 観点 | 内容 |
|------|------|
| **核心メッセージ** | 紹介を出す前に「**ティーアップ（事前の価値付け）**」を行うことが、クライアントの不安（ディフェンダー）をブロックする **スクリーン** となる |
| **悪い状態** | 単に連絡先を繋ぐだけだと、紹介されたメンバーが「手強いディフェンダーと一対一で戦う」状態になり、成約率が上がらない |
| **良い一言の例** | 「次に紹介する A さんは、私が自信を持って推薦するプロフェッショナルです」 |
| **拡大目標との連動** | 現在51名から **75名体制** へ拡大すると、紹介が生まれるパスコースは **2,775通り**（約2.2倍）に拡大 |
| **今週のワーク** | リファーラルを出す際、ただ連絡先を繋ぐのではなく、相手の不安を取り除くティーアップの一言（スクリーン）を必ず添える |
