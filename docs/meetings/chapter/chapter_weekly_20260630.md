**テーマ:** 日本代表チームと BNI パワーチームの構造的類似

| 観点 | 内容 |
|------|------|
| **日本代表の構造** | 普段は別クラブに所属しながら、招集時に「日の丸」のもと一丸となる。BNI チャプターと同じ構造 |
| **パワーチーム** | ストライカー（攻撃）・中盤（パス）・守備がそろって初めて最強チームが完成するように、異業種のプロが組み合わさることでクライアントへの完璧な提案が可能 |
| **拡大戦略** | 現在51名→75名体制を目指す。75名で紹介パスコースは2,775通り（現在の約2.2倍） |
| **ビジター招待の位置づけ** | 「組織への勧誘」ではなく「代表チームへの名誉ある招集」 |
| **今週のワーク** | 自身のビジネスパワーチーム完成のため、「絶対に欠かせない業種・ポジション」を1つ明確にし、そのプロにスカウトをかける |
