# PHASE_168_imanishi_ilc_cost_sanity_check WORKLOG

## 判断ログ

### 費用感の再確認

Phase 167 の Web アプリ費用レンジは、スモールスタートの説明としては使えるが、相手に提示する見積前提としては一部低めだった。

特に、Web アプリでは以下の実装・運用コストが発生する。

- TSV の文字コード・例外データ対応
- 集計条件の差分確認
- Excel / PDF の出力品質調整
- ログイン・権限・履歴
- サーバー運用・バックアップ
- 月次運用中の軽微な修正・問い合わせ対応

そのため、相談メモの費用レンジを安全側に上方修正した。

### 上方修正後の考え方

Web検証は 10〜20万円、Webアップロード最小版は 80〜150万円、加盟店向けWebレポート生成版は 150〜250万円を目安とした。

送信管理まで含める場合は、誤送信防止・宛先管理・送信履歴・再送対応など業務リスクが増えるため、400〜700万円のレンジにした。

### ランニング費用

サーバー代だけでなく、保守・軽微修正・月次運用見守りを含める前提に改めた。保守は月 5〜10万円程度が妥当と判断した。
