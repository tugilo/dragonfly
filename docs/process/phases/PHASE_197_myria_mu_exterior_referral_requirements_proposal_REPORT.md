## Summary
株式会社Myria-mu の外構工事紹介・施工店送客・インフルエンサー報酬管理システムについて、クライアント提案書の前段となる要件定義書を `docs/requirements/myria_mu_exterior_referral_system_requirements.md` として作成した。

文書は、経営者向けにプロジェクト概要、現在の課題、システム化の目的、As-Is / To-Be 業務フロー、ペルソナ、機能要件、画面一覧、権限設計、データ管理項目、将来拡張、スコープ外、概算見積対象範囲を整理している。

通知は、案1「メール通知のみ」と案2「公式LINE通知」に分け、公式LINE通知は初期構築では追加オプション扱いとした。概算予算はユーザー指示に合わせて 150万円〜180万円とし、初期構築範囲は紹介・案件・契約・完了・報酬管理の中核機能へ絞った。
