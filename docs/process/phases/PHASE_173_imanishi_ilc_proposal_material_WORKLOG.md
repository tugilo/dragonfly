# PHASE_173_imanishi_ilc_proposal_material WORKLOG

## 判断ログ

### 提案資料の置き場

今西さんへ共有する可能性があるため、相談メモではなく `docs/proposals/` に配置した。`docs/consultations/ILC/` は内部の調査・検討ログ、`docs/proposals/` は外部共有用の提案資料という役割分担にした。

### 構成

提案資料は、金額表から始めるのではなく、結論と削減根拠から始めた。経営者目線では「何ができるか」より先に「どれくらい手間が減り、投資回収できるか」が重要なため。

### 提案の中心

フル機能ではなく、AI駆動Web MVP を中心にした。ログイン・送信管理・高度な履歴は初期スコープから外し、まず TSV アップロードから集計表出力までを実用化する方針にした。
