# PHASE_169_imanishi_ilc_roi WORKLOG

## 判断ログ

### ROI の扱い

費用だけでは経営判断が難しいため、作業時間削減と投資回収期間を相談メモに追加した。

実際の月次作業時間は未確認のため、20時間・40時間・60時間・80時間の仮説ケースを置き、削減時間と時給換算の月次削減額を示した。時給単価は 3,000円と 5,000円の2パターンにした。

### 回収期間

プラン別に、月次削減額 10万円・20万円・30万円の場合の投資回収期間を整理した。保守費用がある場合は、月次削減額から保守費を差し引いた純削減額で見る必要があることも明記した。

### 経営者向けの判断軸

単純な作業時間削減に加え、属人化低減、転記ミス低減、提出遅れ・作り直し削減、過去データ再出力、担当者の時間を分析・改善提案に回せることを効果として整理した。
