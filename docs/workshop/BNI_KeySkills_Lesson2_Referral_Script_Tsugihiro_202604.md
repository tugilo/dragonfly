## ■紹介文（完成形）

**AI業務改善システム構築の次廣さん**（tugilo）は、**事務や現場の仕事が Excel・手書き・同じ作業を二回やる運用でパンクしそうな中小の会社や個人事業主**向けに動いている人です。人柄は穏やかで、**いきなり売り込まず、相談から入って「なんとなく不便」を一緒に言葉にする**のが得意です。**働き方を根こそぎ変えなくてよい**ようにしながら、**相手に合ったやり方**に整えていきます。**まだ売上はあるが、事務・現場が回り切れない経営者**、**一人で営業も事務も抱えて限界の個人事業主**、**何から手をつければいいか迷っている**方に会うと思います。

## ■事例と効果（紹介の「刺さり」を足す用・匿名の型）

※実在の固有名は出さない。**場面 → やったことのイメージ → 変化** が一言で言えれば十分。

1. **個人事業主** — 請求や入金の記録を、紙と表計算の**両方**でつけて夜にかかっていた。**書く場所を一つにまとめた**ところ、**月末の確認が短くなり、抜けミスも減った**、というパターン。  
2. **小さな会社** — 受けた注文の内容を、チャットと表計算の**二重**で管理していて、どちらが正か分からなくなりがち。**流れを一本化**して、**担当が休んでも続けられる形**になった、というパターン。  
3. **よくある成長途中の現場** — 毎晩、帳簿や発注データの**写し直し**が残業の原因になっていた。**手順を分けて、つなぎ目だけを短くした**ところ、**残業が目に見えて減った**、というパターン。

**効果を短く言うなら:** **時間に余裕が出る**・**ミスや手戻りが減る**・**同じ作業を二回やらなくてよくなる**・**一人社長でも夜が少し楽になる**、など。
