## ■①どんな課題を解決できるか

・**事務や現場の仕事**が、**Excel・手書き・同じ作業を二回やる運用**などで重くなっている。  
・**Excel管理が限界**に近い／限界になってきた（回しきれない、誰かいないと止まる、といった状態を含む）。  
・そのままだと**時間も人手もいつも足りない**感じが続きやすい。  

---

## ■②どのように助けるか

・**AIの助けも借りながら**、手間のかかる仕事を**小さな手順に分けて整理**する。（上の原文では短く言っている部分と同じ意味。）  
・整理したうえで、**その会社に合ったやり方・仕組み**をつくる。  
・**現場の人の動きは、無理に変えない。**  
・**気づいたら時間に余裕が出ている**感じに近づける。（原文の「時間が生まれている」と同じイメージ。）  
・人との関わり方は、**いきなり売り込まず、相談から入って「なんとなく不便」を一緒に言葉にする**スタイル（紹介文 Script と同じ）。  
・ウィークリーで短く名乗っている部分は、**時間に余裕が出るよう仕事の回し方を一緒に整える人**という意味合い。カテゴリー名は **AI業務改善システム構築**（名簿どおり）。  
・**口頭や第三者紹介**では、**困りごとの具体＋そのあとの変化（効果）**を一言足すと刺さりやすい（匿名の型は Script「■事例と効果」）。

## ■③誰を助けるビジネスか

・**ウィークリー（§2.1）でお願いしている紹介：** **Excel運用が限界に近い会社様を顧問先・クライアントで見ている士業・コンサルの方** とお繋ぎいただけると助かる。（締めの一文どおり。短縮稿では「現場がExcel限界の会社を周りで見ている士業・コンサル…」に言い換え可。）  
・**最終的に仕事として助けるのは：** **Excel管理が限界に近い会社**で、その状況を抱えている人。  
・**経営者**、または**現場の仕事の流れを握っている責任者・管理者**。（ウィークリー本文に業種・会社規模は出てこないので、ここでも足さない。）  
・**個人事業主** — 売上はあるが**事務が一人に集中**して限界、同じ作業を二か所でやっている、など。（WSフィードバック。**ウィークリー本文に明示はない**が、紹介設計では同等に扱う。）  
・**紹介してほしい人の具体例**（表現は Script と同一ノリ）：**注文・請求・事務まわり**が回り切れない人、**一人社長・個人事業主**で事務がパンク寸前の人、**どこから直せばいいか**決められない人、**市販のソフトや外のサービスが現場に合わない**会社、**会社の仕事をネットやパソコンでまとめたいがまだはっきりしない担当者**、**相談しながら**少しずつ整えたいチーム — 詳細文言・**事例と効果**は [Script](./BNI_KeySkills_Lesson2_Referral_Script_Tsugihiro_202604.md) の「■紹介してほしい人」「■事例と効果」。
