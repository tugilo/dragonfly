# Genspark用プロンプト: AiLC 月次集計表作成 自動化提案スライド

以下を Genspark に貼り付けて、提案スライドを作成する。

## プロンプト

あなたは、BtoB業務改善・AI活用・Webアプリ提案に強い、経営者向け提案資料のプロフェッショナルです。

以下の提案メモをもとに、今西さん向けの分かりやすい提案スライドを作成してください。

### スライドの目的

AiLC の月次集計表作成業務について、現状の手作業を AI駆動Webアプリでどのように自動化できるかを説明し、まずは小さく検証・MVP開発へ進む判断材料を提示する。

### 想定読者

- 今西さん
- AiLC 側の経営者・管理者
- システム担当者

読者は技術者とは限らないため、専門用語を使いすぎず、経営判断に必要な「手間削減」「費用対効果」「段階的な進め方」が伝わる構成にしてください。

### トーン

- 売り込みすぎず、落ち着いた提案
- 「いきなり大きく作る」ではなく「小さく検証して、効果が見えたら拡張」
- AIを魔法のように見せず、AI駆動開発で検証・改善を速く回せることを伝える
- 金額よりも、業務削減・属人化解消・投資回収の見通しを重視

### デザイン方針

- 白背景または薄いグレー基調
- アクセントカラーは青系またはグリーン系
- 経営者が短時間で理解できる、余白のあるレイアウト
- 1スライド1メッセージ
- 表や数値は見やすく、情報を詰め込みすぎない
- 必要に応じて、Before / After、3ステップ、ROI、削減率を図解する

### 必ず入れる重要メッセージ

1. TSV取得後の加工〜集計表作成部分は、自動化余地が大きい。
2. 確認済みデータは、チカラグループ15ファイル、ファボローネ36ファイル、合計51 TSV、76,628行。
3. TSVの列構造が揃っているため、プログラムによる一括処理に向いている。
4. TSV取得後の加工〜集計表作成に絞ると、80〜95%程度の作業削減が見込める。
5. 月次運用全体で見ても、60〜85%程度の削減が見込める。
6. tugilo はAI駆動開発で、TSV解析・集計ロジック試作・画面プロトタイプを短期間で検証できる。
7. 最初から大きなシステムを作らず、AI検証 → Web MVP → レポート生成版の順で進める。
8. 初回提案の中心は、80〜90万円のAI駆動Web MVP。
9. 月40時間の作業が4〜8時間程度まで減るなら、月32〜36時間の削減になる。
10. 投資判断には、現在の月次作業時間・作業人数・加盟店数・送付形式の確認が必要。

## 元資料

以下の内容を根拠として使用してください。必要に応じて要約し、スライド向けに再構成してください。

```markdown
# AiLC 月次集計表作成 自動化提案メモ

作成日: 2026-06-01 23:26 JST  
作成: tugilo / 次廣淳  
対象: 今西さん / AiLC 月次集計表作成業務

## 結論

いただいたデータと作業手順を確認した限り、月次集計表作成のうち、TSV取得後の加工〜集計表作成部分はかなり自動化できる可能性が高い。

現在 Excel への貼り付け、前回データ削除、日付補完、変換、転記を繰り返している部分は、プログラムで一括処理しやすい構造。

最初から大きなシステムを作るのではなく、まずは AI駆動で小さく検証 → Web MVP → レポート生成版の順で進めるのが現実的。

## 検証で分かったこと

- 対象データ: チカラグループ / ファボローネ
- TSV数: チカラグループ 15ファイル、ファボローネ 36ファイル、合計 51ファイル
- データ行数: 合計 76,628 行
- 文字コード: cp932 で読み取り可能
- TSV列: 店名, 日付, category, action, label, 件数, 時, 分, 秒
- 現行作業: TSVをExcelへ貼り付け、削除、日付補完、変換、転記、最終表作成

## 自動化できること

- TSVファイルの読み込み
- 文字コード変換
- 店舗別・日別・項目別の集計
- 日付欠損の補完
- Excel / CSV への出力
- 加盟店向けレポートの形式調整

初期段階では、TSVをアップロードすれば集計表が出るところに絞るのが費用対効果の面で良い。

## 削減効果

TSV取得後の加工〜集計表作成に絞ると、80〜95%程度の作業削減を見込める。

- 20時間/月 → 2〜4時間/月
- 40時間/月 → 4〜8時間/月
- 60時間/月 → 5〜10時間/月
- 80時間/月 → 6〜12時間/月

TSV取得・送付確認まで含めた月次運用全体でも、60〜85%程度の削減が見込める。

## 費用対効果

月40時間の作業が4〜8時間まで減る場合、月32〜36時間の削減。

- 3,000円/時換算: 約9.6〜10.8万円/月
- 5,000円/時換算: 約16〜18万円/月

AI駆動Web MVP が 80〜90万円であれば、概ね数ヶ月〜1年弱で回収できる可能性がある。

## 提案プラン

### プランA: AI検証プラン

概算: 10〜20万円（税別）

代表店舗のTSVを使い、現行Excelと主要数値が合うかを確認。

### プランB: AI駆動Web MVP

概算: 80〜90万円（税別）

ブラウザからTSVをアップロードし、店舗別・日別・項目別に集計したExcel / CSVを出力する最小実用版。

含めるもの:
- TSVアップロード
- 店舗別・日別・項目別集計
- 日付欠損補完
- Excel / CSV 出力
- 最小限の画面

初期スコープから外すもの:
- 本格的なログイン機能
- 送信管理
- 宛先管理
- 高度な履歴管理

### プランC: AI駆動レポート生成版

概算: 160〜180万円（税別）

加盟店に送る資料として見やすいExcel / PDFレポートを、Web上で自動生成する版。

## 推奨する進め方

Step 1: 代表店舗でAI駆動の集計ロジックを検証（10〜20万円）
Step 2: TSVアップロードから集計表出力までのWeb MVP（80〜90万円）
Step 3: 加盟店向けExcel / PDFレポート生成（追加80〜90万円）

初回は、Step 1 または Step 1+2 までに留めるのが良い。

## tugilo のAI駆動開発

tugilo では、AI駆動開発により、TSV解析、集計ロジックの試作、画面プロトタイプ作成、Excel / PDF 出力の検証、複数レポート案の比較、現行Excelとの数値照合を短期間で進められる。

AIを使う目的は、単に安く作ることではなく、早く試し、数字で確認し、投資判断の失敗リスクを下げること。
```
