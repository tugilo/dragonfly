## 抜き打ちテスト 模範解答

### 1. ウィークリープレゼンテーションに含める必要がある4つの要素は何ですか？

ウィークリープレゼンテーションには、`私は誰か`、`何をしているか`、`誰を紹介してほしいか`、`どう紹介してほしいか` の4つの要素が必要だと理解しています。  
つまり、カテゴリーと提供価値を明確にし、理想的な紹介先と紹介の切り出し方まで具体的に伝えることが大切です。

### 2. プランニング用ドキュメントを作成していた際に、分からなかったところはありましたか？ それを書き留めてましたか？

はい、ありました。特に、理想的な紹介先をどこまで具体的に書くか、そして紹介してもらう相手にどのように声をかけてもらうかは迷いやすい部分でした。  
そのため、曖昧なままにせず、困っている状況、業種、相手の特徴、紹介の一言などをメモしながら埋めるようにしました。

### 3. ウィークリープレゼンテーションは（具体的に）VCPプロセスのどこに分類されますか？ なぜそこであり、他の2つではないのでしょうか？

ウィークリープレゼンテーションは、主に `Visibility` に分類されると考えています。  
理由は、毎週の短いプレゼンを通じて、自分の仕事、強み、探している紹介先をメンバーに知ってもらう活動だからです。  
`Credibility` は1to1や実績、約束を守る行動などを通して深まるもので、`Profitability` は実際に紹介や成約につながった段階です。したがって、ウィークリーはまず認知の段階にあると理解しています。

### 4. 信頼を築くのにウィークリープレゼンテーションはどのように役立つと思いますか？

ウィークリープレゼンテーションは、毎週一貫して自分の専門性や価値を伝えることで、周囲に `この人は何の専門家か` を明確にします。  
内容が具体的で、毎回ぶれず、相手に分かりやすいほど、安心して紹介しやすい人だと感じてもらえます。  
つまり、ウィークリーは信頼を一度に作るものではなく、認知を積み重ね、その結果として信頼の土台を作るものだと思います。

### 25秒版の運用メモ

- この版をDragonFly例会用のFix案とする
- 25秒枠には収まりそう
- 毎週同じ内容に固定しすぎず、2週間に一度または月ごとに業種を変えると、紹介の入口が広がる
- 例: 建設業・製造業 → サービス業 → 複数拠点運営 → 士業の顧問先 など

### 業種差し替え候補

tugiloとして関わってきた領域を踏まえると、業種は固定せずに複数パターンで回せる。

| 候補 | 刺さる困りごと | ウィークリーでの言い方 |
|------|----------------|------------------------|
| 建設業 | 見積・現場・請求がバラバラ、担当者しか分からない | 建設業で、見積・現場・請求がExcelや手作業でバラバラな会社様 |
| 製造業 | 工程・在庫・受発注・実績がExcel管理 | 製造業で、Excel管理が増えすぎて現場と事務の情報が合わない会社様 |
| 解体工事業 | 問い合わせから見積、請求までの流れが追いづらい | 解体工事業で、案件の見落としやステータス管理に困っている会社様 |
| サービス業 | 予約・顧客対応・請求・実績管理が分断 | サービス業で、予約や顧客管理をLINEやExcelで分けて管理している会社様 |
| 飲食・FC本部 | 加盟店管理、注文、実績把握に手戻りがある | 飲食FC本部で、店舗とのやり取りや注文管理に手間がかかっている会社様 |
| 複数拠点運営 | 拠点ごとにExcelやメールが分かれ、本部集計が遅い | 複数拠点で、数字や実績を本部が集めるのに時間がかかっている会社様 |
| イベント・キャンペーン運営 | 参加者管理、問い合わせ、集計が手作業 | イベント運営で、参加者が増えるほど管理が大変になる会社様 |
| 予約型ビジネス | 電話・メール・LINE予約が混在し、取りこぼしがある | LINE予約や顧客対応をもっと楽にしたいサービス業の会社様 |
| 士業・コンサルの顧問先 | 顧問先の業務改善ニーズを見ている | 顧問先で、Excel管理や属人化に困っている会社様を見ている士業・コンサルの方 |
| 中小企業全般 | 人を増やす前に仕組みを整えたい | 忙しすぎて改善に手が回らない中小企業の経営者様 |

### 業種ローテーション案

- 第1週: 建設業・製造業
- 第2週: サービス業・予約型ビジネス
- 第3週: 複数拠点運営・FC本部
- 第4週: 士業・コンサルの顧問先
- 必要に応じて: 解体工事業、イベント運営、LINE予約など具体案件に寄せる

### 別項: ウィークリープレゼンのコツ

- 誰が聞いても、何をしている人かすぐ分かる言い方にする
- 専門用語より、相手が思い浮かべやすい困りごとの言葉を使う
- 聞いた人が「この人に合いそう」と思えるように、紹介先を具体的にする
- リファーラルを出しやすくするには、誰に声をかければよいかまで伝えるのがコツ
- BOフィードバック反映: 業種を先に出す、属人化は具体例で言う、次廣に任せる理由を一言入れる
