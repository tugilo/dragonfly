# メインプレゼンテーション「抜き打ちテスト！」— 解答

**問題出典:** *Main Presentation Participant Guide*（メインプレゼンテーション 受講者ガイド）抜き打ちテスト（BNI教材）。

**正解（受講者ガイド基準・以下をSSOTとする）:**

| # | 問題（要約） | **正解** |
|---|----------------|----------|
| **1** | メインプレの何週間前に、仲間のBNIメンバーに知人を招待するよう依頼するか | **4週間前** |
| **2** | プレゼンの何週間前に、10〜15人のクライアント・友人・家族を招待するか | **3週間前** |
| **3** | 他のBNIメンバーに、あなたのビジネスへの疑問・質問があるか尋ねるのに適したタイミング | **6週間前** |
| **4** | メインプレのミニプレビューのためにウィークリープレゼンを使うのに適したタイミング | **5週間前** |
| **5** | 招待した人々を何週間前にフォローアップするか（あなたとメンバー） | **2週間前** |

---

## 時系列で並べると（メモ）

遠い順から近い順に整理すると、次のようになる（各項目がガイド上の「その週にやる」目安）。

| いつ | やること |
|------|----------|
| **6週間前** | メンバーに **疑問・質問がないか** 尋ねる（準備の最初の打ち合わせ・ヒアリング） |
| **5週間前** | **ウィークリー**で **ミニプレビュー**（メインプレの予告・一端を見せる） |
| **4週間前** | 仲間メンバーに **知人招待を依頼** |
| **3週間前** | **10〜15人**（クライアント・友人・家族）への **招待** |
| **2週間前** | 招待した人への **フォローアップ** |

※以前の版では一般的な海外ブログ等に基づく別の週数をメモしていたが、**上表の5つの数字が受講者ガイドの正解**。
