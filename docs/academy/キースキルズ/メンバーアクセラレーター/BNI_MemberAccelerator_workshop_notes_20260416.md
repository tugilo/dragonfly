### SMART ゴール（講義メモ）

| 文字 | 意味（ガイド） | 自分の言葉で |

| S | Specific（具体的） | |

| R | Realistic（現実的） | |
| T | Time-bound（期限） | |

### 数値例の考え方（ガイドの例）

スライドの式: **求める年間リファーラル数** ＝ 年間売上 ÷ 平均単価／**得たい総リファーラル数** ＝ 求める年間リファーラル数 ÷ 成約率。

### 1年目の目標（自分用・2026-04-16 反映）

※「75%」は **成約率** として計算（別の意味なら書き換え）。

| 項目 | 値 | メモ |
|------|-----|------|
| チャプターから得たい年間の売り上げ | **1,000万円** | 1年目 |
| 最も欲しいリファーラルの平均単価 | **100万円** | |
| 求める年間リファーラル数（成約分） | **10件** | 1,000万 ÷ 100万 |
| 成約率 | **75%** | |
| 得たい総リファーラル数（紹介の機会が必要な件数） | **約13.3件**（月 **約1.1件**） | 10 ÷ 0.75 ＝ 13.333…／年12で割る |

**要点:** 成約までに紹介が **4回に3回** つながる想定なので、「成約10件」には紹介ベースで **約13〜14件** のリファーラル機会が必要、という読み方。
