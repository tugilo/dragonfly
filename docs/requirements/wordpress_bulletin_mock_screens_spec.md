# モック作成 — 画面一覧と画面ごとの要素

**目的**: 西浦氏・平岡氏レビュー用モック（デザイン or ワイヤー）の**作るべき画面**と、**各画面に必ず入れるUI要素**を**見落としなく**定義する。  
**根拠文書**: `docs/requirements/wordpress_bulletin_and_plugin_reorganization_requirements.md`  
**想定**: **案件A（会員向け投稿・案件共有）** 中心。**案件B** は図1枚 or 1画面の**補足**で可。

**表記ルール**

- **必須**: 今回の合意形成に**最低限**欲しい。  
- **条件付き**: 仕様分岐（承認有無、タグ利用等）で**出し分け** or **注記**で表現。  
- **任意**: 初期見積外でも**議論**に使えるとよい。

---

## 0. モック前に固定しておくとよい前提（1枚の注記用）

| 項目 | モック上の扱い |
|------|----------------|
| 会員専用エリア内の導線 | 既存会員メニューから「◯◯（掲示板の呼称）」に**遷移**する前提で1パス**示す** |
| 掲示板の呼称 | 仮名で可（**正式名称**は後決め） |
| デスクトップ + スマホ | **一覧**は**幅768px前後**の**1画面**、可能なら**主要2画面**は**縦スクロール**含め**簡易**で可 |

## 1. 会員向け — 画面一覧

| ID | 画面名 | 目的 | 対応する主な合意事項 |
|----|--------|------|------------------------|
| M1 | 投稿**一覧** | 絞り込み・検索の起点 | カテゴリUI、一覧カード、空状態、検索 |
| M2 | 投稿**詳細** | 本文 + コメント | 表示項目、承認中の扱い、コメント有無 |
| M3 | 投稿**新規作成** | 会員が投稿する導線 | 入力項目、プレビュー、送信後 |
| M4 | 投稿**編集**（**条件付き**） | 本人のみ等の**編集** | 編集**をやる／やらない**の**分岐** |
| M5 | **検索結果**（**推奨**） | 検索後の導線の明確化 | 0件、複数件、**サイドバー**との関係 |
| M6 | **承認待ち**の見え方（**条件付き**） | 承認**あり**案の**画面差分** | 投稿者/管理者**どちら向け**か**注記** |

## 2. 画面別 — 含めるUI要素

### M1 投稿一覧

| 区分 | 要素 | 必須/任意 |
|------|------|-----------|
| レイアウト | **2カラム**（**左: カテゴリ**、**右: 一覧**）想定。スマホでは**カテゴリ**の**畳み** or **上段** | 必須 |
| 左カラム | **カテゴリ**一覧、**件数**（仮数で可）、**「すべて」**、**階層**（1段階は必須、多段階は要確認） | 必須 |
| 左カラム | **現在**の**選択中**カテゴリの**強調** | 必須 |
| 右 | **キーワード検索**（**プレースホルダ**文言入り） | 必須 |
| 右 | **投稿カード/行**: **タイトル**、**日付**、**抜粋1〜2行**、**カテゴリ**（バッジ）、**投稿者** | 必須 |
| 右 | **新着順**等の**並び**（仮） | 任意 |
| 右 | **ページネーション** or **「もっと見る」** | 必須 |
| 導線 | **「新規投稿」**ボタン（**権限ある人のみ**可と**注記**可） | 必須 |
| 状態 | **0件**（**このカテゴリにはまだ**等） | 必須 |
| グローバル | パンくず、ページタイトル（H1 相当） | 推奨 |

---

### M2 投稿詳細

| 区分 | 要素 | 必須/任意 |
|------|------|-----------|
| ヘッダ | **タイトル**、**日付**、**カテゴリ**、**投稿者** | 必須 |
| 本文 | **本文**（**長文**想定で**スクロール**） | 必須 |
| 条件 | **承認前** / **下書き**の**見え方**（**承認なし**案なら**注記**「このブロックは出ない」） | 条件付き |
| コメント | **コメント一覧**（**投稿者名**、**日時**、**本文**） | 必須 |
| コメント | **入力**＋**送信**（**未ログイン**は**M7**） | 必須（コメント**なし**運用の場合は**方針**を**1行**注記） |
| 導線 | **一覧に戻る**、**前後の投稿**（**任意**） | 推奨 / 任意 |
| SEO | 共有メタは**今回**は**出さない** or **注記** | 任意 |

---

### M3 投稿新規作成

| 区分 | 要素 | 必須/任意 |
|------|------|-----------|
| 項目 | **タイトル**（**必**）、**本文**（**必**）、**カテゴリ**（**必**、**セレクト** or **ラジオ**想定） | 必須 |
| 項目 | **タグ**（**使う**場合のみ） | 条件付き |
| 項目 | **抜粋**、**アイキャッチ** | 任意（**初期**は**出さない**でよい**旨**注記可） |
| 操作 | **下書き保存**、**投稿（公開依頼）** 等、**承認**の**有無**で**ラベル**を**変える** | 条件付き |
| 補足 | **投稿ガイド**（禁止事項、個人情報**注意**）**テキスト**枠 | 推奨 |
| 送信中 | **送信中**の**抑止**（**二重押し**）**は**注記**で可** | 推奨 |
| 完了 | **成功後**の遷移先（**詳細** or **一覧**）**を**1案** | 必須 |

---

### M4 投稿編集（条件付き）

| 含める | 内容 |
|--------|------|
| M3 と**同じ**項目＋**「削除」**の**要/不要** | 要否は**要合意**。**不要**なら**本画面**は**作らない**＋**注記** |
| **編集**できるのは**本人のみ** 等、**図**外に**1行** | 必須（**編集**を**やる**場合） |

---

### M5 検索結果

| 要素 | 備考 |
|------|------|
| **キーワード**表示、**件数** | 必須 |
| 一覧部は**M1**の**右カラム**と**同系** | 必須 |
| **0件** | 必須 |
| 左カラム**カテゴリ**の**併用**（**絞り直し**） | 推奨 |

---

### M6 承認フロー（条件付き）

- **片方**でよい: **A)** 会員の「**承認待ち**」**一覧** or **B)** 管理側の**簡易**一覧。  
- **要素**: **ステータス**、**日付**、**差し戻し**有無**は**「**将来**」**注記**可。

---

### M7 境界 — 未ログイン・権限不足

| パターン | 見せる内容 |
|----------|------------|
| 会員専用URLを**会員外**が開いた | **ログイン誘導**、**趣旨1文** |
| **投稿**権限**なし** | **閲覧のみ**、**投稿ボタン****非表示** or **不可****メッセージ** |

---

### M8 管理者 — カテゴリ管理（任意）

- **カテゴリ名**の**追加**、**親子**、**表示順**、**不要なら**「**会員用**は**このツリー**」**説明**のみ。

---

### B1 プラグイン整理（1枚図）

- **入れる語**: 棚卸し、**停止/保留/維持**、**4領域**（トップ・会員・掲示板・プロフィール）、**一気に消さない**、**本番**は**段階＋予備**（**要約**で可）。

## 3. モック作成後のレビュー観点（チェックリスト）

- [ ] **スマホ**で**カテゴリ**が**辿れる**か  
- [ ] **承認**の**有無**で、**M3/M2/M6**の**食い違い**がないか  
- [ ] **誰が投稿**・**誰がコメント**の**方針**と**導線**が**一致**しているか  
- [ ] **既存会員**メニューから**物理的に****どこを押す**か**示した**か
